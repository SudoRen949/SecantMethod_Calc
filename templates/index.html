01 Mathematical Discussion

The Secant Method is an open root-finding algorithm designed to avoid manual differentiation workflows by approximating tangents across sequential secant intercepts.

Requested Structural Core Formula Formulation:

$$x_c = x_b - \frac{f(x_b)(x_a - x_b)}{f(x_a) - f(x_b)}$$

Error Evaluation Mechanism

To determine system resolution convergence thresholds, the execution loops use an explicit estimated relative error formula based on consecutive structural approximations: $$e = \left| \frac{x_{\text{new}} - x_{\text{old}}}{x_{\text{new}}} \right| \le \epsilon$$ Where $x_{\text{new}} = x_c$ and $x_{\text{old}} = x_b$. The calculation routine ceases execution the moment $e \le \epsilon$.

02 Step-by-Step Solved Examples

Example 1: Polynomial Function

Find the real root of $f(x) = x^3 - x - 1 = 0$ given initial guesses $x_a = 1$ and $x_b = 2$. Target tolerance $\epsilon = 0.000001$.

1st Iteration solution
$f(x_a) = 1^3 - 1 - 1 = -1$
$f(x_b) = 2^3 - 2 - 1 = 5$
$x_c = 2 - \frac{(5)(1-2)}{(-1)-(5)} = 1.16666$
$f(x_c) = 1.16666^3 - 1.16666 - 1 = -0.57870$
$e = | \frac{1.16666 - 2}{1.16666} | = 0.71428$

Example 2: Transcendental Function

Find the real root of $f(x) = \cos(x) - x \cdot e^x = 0$ using initial guesses $x_a = 0$ and $x_b = 1$.

1st Iteration solution
$f(x_a) = cos(0) - 0 \cdot e^0 = 1$
$f(x_b) = cos(1) - 1 \cdot e^1 = -2.17798$
$x_c = 1 - \frac{(-2.17798)(0-1)}{1-(-2.177980)} = 0.31466$
$f(x_c) = cos(0.31466) - 0.31466 \cdot e^{0.31466} = 0.51987$
$e = | \frac{0.31466 - 1}{0.31466} | = 2.17798$
